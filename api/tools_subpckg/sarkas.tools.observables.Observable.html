sarkas.tools.observables.Observable¶

class sarkas.tools.observables.Observable[source]¶

Parent class of all the observables.

Variables

  • ~Observable.dataframe (pandas.DataFrame) – Dataframe containing the observable’s data averaged over the number of slices.

  • ~Observable.dataframe_acf (pandas.DataFrame) – Dataframe containing the observable’s autocorrelation function data averaged over the number of slices.

  • ~Observable.dataframe_acf_slices (pandas.DataFrame) – Dataframe containing the observable’s autocorrelation data for each slice.

  • ~Observable.dataframe_slices (pandas.DataFrame) – Dataframe containing the observable’s data for each slice.

  • ~Observable.max_k_harmonics (list) – Maximum number of \(\mathbf{k}\) harmonics to calculate along each dimension.

  • ~Observable.phase (str) – Phase to analyze.

  • ~Observable.prod_no_dumps (int) – Number of production phase checkpoints. Calculated from the number of files in the Production directory.

  • ~Observable.eq_no_dumps (int) – Number of equilibration phase checkpoints. Calculated from the number of files in the Equilibration directory.

  • ~Observable.no_dumps (int) – Number of simulation’s checkpoints. Calculated from the number of files in the phase folder.

  • ~Observable.dump_dir (str) – Path to correct dump directory.

  • ~Observable.dump_step (int) – Correct step interval. It is either sarkas.core.Parameters.prod_dump_step or sarkas.core.Parameters.eq_dump_step.

  • ~Observable.no_obs (int) – Number of independent binary observable quantities. It is calculated as \(N_s (N_s + 1) / 2\) where \(N_s\) is the number of species.

  • ~Observable.k_file (str) – Path to the npz file storing the \(k\) vector values.

  • ~Observable.nkt_hdf_file (str) – Path to the npy file containing the Fourier transform of density fluctuations. \(n(\mathbf k, t)\).

  • ~Observable.vkt_file (str) – Path to the npz file containing the Fourier transform of velocity fluctuations. \(\mathbf v(\mathbf k, t)\).

  • ~Observable.k_space_dir (str) – Directory where \(\mathbf {k}\) data is stored.

  • ~Observable.saving_dir (str) – Path to the directory where computed data is stored.

  • ~Observable.slice_steps (int) – Number of steps per slice.

  • ~Observable.dimensional_average (bool) – Flag for averaging over all dimensions. Default = False.

  • ~Observable.runs (int) – Number of independent MD runs. Default = 1.

  • ~Observable.multi_run_average (bool) – Flag for averaging over multiple runs. Default = False. If True, runs needs be specified. It will collect data from all runs and stored them in a large ndarray to be averaged over.
